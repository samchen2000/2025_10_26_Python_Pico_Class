## VScode 快捷鍵整理

多行註解
```
Shift + Alt + A
```

選取整行
```
Ctrl + L
```
查詢變數
```
對變數按右鍵 + 預覽變數
```
程式碼摺疊
```
Ctrl + K  -> Ctrl + [數字]   // 摺疊層級

Ctrl + K  -> Ctrl + 0       // 摺疊所有程式碼

Ctrl + K  -> Ctrl + J       // 取消摺疊
```
手動選取多個位置
```
Alt + [click]
```
一次選取檔案內所有相同的文字
```
Ctrl + Shift + L
對目標按右鍵 + 變更所有發生次數
```
智慧擴展 / 縮減選擇
```
Alt + Shift + ← →
```
格式化程式碼
```
Alt + Shift + F
```
快速開啟檔案
```
Ctrl + P
```
選取相同字串
```
Ctrl + D
```
單行註解
```
Ctrl + /
```
跳到該檔案的某行
```
Ctrl + G
```
目前游標所在行，向上或向下移動
```
Alt + ↑ ↓
```
目前游標所在行，複製並移動
```
Alt + Shift + ↑ ↓
```
分割編輯視窗
```
Ctrl + [數字]
```
開啟終端機
```
Ctrl + `
```
建立新終端機分頁
```
Ctrl + Shift + `
```
搜尋目前檔案
```
Ctrl + F
```
搜尋全部檔案 / 取代
```
Ctrl + Shift + F
```
