# Streamlit MQTT 監控應用程式 - 產品需求文件 (PRD)

## 專案概述

建立一個基於 Streamlit 的 Web 應用程式，用於即時監控和視覺化 MQTT 訂閱的感測器數據。

## 系統角色

- **應用程式角色**：MQTT 訂閱者（Subscriber）
- **功能定位**：數據監控與視覺化儀表板

## 使用者介面需求

### 顯示項目

1. **電燈狀態顯示**
   - 顯示電燈的開關狀態（開/關）
   - 建議使用視覺化指示器（如圖示或狀態燈）

2. **環境數據顯示**
   - **客廳溫度**：即時顯示當前溫度數值
   - **客廳濕度**：即時顯示當前濕度數值

3. **數據視覺化**
   - 使用圖表（Chart）展示溫度和濕度的歷史趨勢
   - X 軸：日期和時間（時序數據）
   - Y 軸：溫度和濕度的數值

## 功能需求

### 核心功能

1. **即時數據顯示**
   - 從 MQTT Broker 訂閱感測器數據
   - 即時更新電燈狀態、溫度、濕度資訊
   - 僅提供顯示功能（唯讀模式）

2. **數據儲存**
   - 將訂閱接收到的所有數據自動儲存為 Excel 檔案
   - 建議包含時間戳記、電燈狀態、溫度、濕度等欄位
   - 確保數據持久化，方便後續分析

## 技術規格

- **開發框架**：Streamlit
- **通訊協定**：MQTT
- **數據格式**：Excel (.xlsx)
- **更新頻率**：即時（根據 MQTT 訊息接收頻率）
